크롬 브라우저 최적화

  • 1) Vegas.js
    플러그인 소스인 js / css파일 연결
    $(class).vegas({slide:[{src:파일},{src:파일}])
  • 2) Javascript를 이용한 mousemove 이벤트
    마우스의 움직임, 즉 좌표값을 얻어내는 e.pageX, e.pageY를 추적,
    이미지의 좌표값을 정확히 입력후 원하는 이벤트 발생
  • 3) Delay 없이 서서히 움직이는 가로 롤링 이미지
    이미지 정렬한 뒤 class 속성값을 넣어줌
    이미지 갯수를 카운터해서 last변수에 저장
    이미지 갯수를 확인 후 테두리 밖으로 벗어나는 갯수면 옆으로 이동
    (매끄러운 전개를 위해 1px씩 왼쪽으로 이동시킴)
    제일 앞의 이미지의 left 값이 이미지의 width 값보다 작으면 제일 뒤로 보내주는 함수 작성
  • 4) Page scroll
    css에 화면에 표출되기 전/후 작성(애니메이션)
    사용자의 모니터 화면높이 + 스크롤이 움직인 높이 확인
    애니메이션 효과를 넣을 DOM 객체 배열
    객체 위치와 높이 비교: 화면에 표출되는 높이 인지 체크
    animatable 클래스를 지우고 클래스 animated 추가

REVIEW

자바스크립트를 많이 사용 해 보지않았기때문에 익숙치 않았지만,
플러그인이나 직접 적용을 해보면서 자바스크립트가 사이트를 구축하는데 굉장히 편리했다.

사이트를 제작하면서 스크립트의 개념을 많이 알아가는 시간이었고
더 많은 공부가 필요하다는것을 느꼈다.
