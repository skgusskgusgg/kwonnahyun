'베스트원테크'라는 예전스타일을 고수하고 있는 기업사이트를 찾았고,
가장 기본적인 디자인으로 리뉴얼하기에 적합하다는 생각이 들었다.

또한 사이트가 사용하고있는 퍼블리싱이 많지 않았기 때문에

크롬 브라우저 최적화

  • 1) Flexible Box
    예전에 많이 사용되던 grid대신에 좀 더 접근하기 편해진 flex를
    이용해 메뉴 부분을 자연스럽게 옆으로 배치 시켜줌.
  • 2) Image Size
    이미지의 원본을 훼손시키지 않기 위해 .box를 만들어 준뒤
    max-width:100%;를 주고 이미지 훼손 방지.
  • 3) hover
    사용자의 경험을 통해 클릭해 주어야 할 부분은 명확하게 나타내
    주기 위해 hover 적용.
  • 4) height:-& -> -vh
    각 영역을 확실히 분단해주기 위해 임의의 %값이 아닌
    viewport height값을 적용.
  • 5) swiper js
    슬라이더 이미지를 사용하기 위해 swiper js사용.
    -> js파일을 연결할 때 js파일을 가장 먼저,
    내가 작성한 js파일을 그 다음에 작성해야 한다는 것을 알게 됨.
    -> script작성시,
    id, class값의 표기부호를 값에 맞게 작성해 주어야 함.

REVIEW

웹 페이지를 처음 직접 구축했을때 내가 어느정도 배움을 가지고
했다고 느꼈음에도 다시 오류가 나는 부분이 많았다.
구조부터 탄탄히 체계적으로 구축한다면, css, js등을 이용할때 헤매지 않고
훨씬 어렵지 않게 웹페이지가 제작되는것을 알게되었다.

자꾸 반복해서 실수하는부분은 한번 더 체크한 뒤에 여러 방법을 시도하여
